# 前缀和

## 前缀和定义

有一个数列$a_1\ a_2\ a_3\ …… a_n$

它的前缀和$S_i\ = a_1\ + a_2\ + …… +\ a_i$

注意：下标从1开始

## 如何求前缀和Si

用for循环

```C++
s[0] = 0 
for (int i = 1; i <= n; i ++ )
    s[i] = s[i - 1] + a[i];
```

- 把前缀和S[0]定义成0，目的：

  为了省去处理边界的步骤，如要计算区间[1, 10]中所有数的和，可以直接算S[10]-S[0]。正常计算[1, 10]所有数的和，直接S[10]就可以算，就是为了统一写成S[r] - S[l-1]的形式。

## 前缀和的作用

**算出区间[l, r]中所有数的和**

### 暴力方法

```C++
sum = 0
for (int i = l; i <= r; i ++ )
    sum += i
```

时间复杂度是o(n)

### 前缀和方法

```C++
S[r] - S[l - 1]
```

时间复杂度是o(1)

S[r]：前r个数的和，$S[r] = a_1 + a_2+……+a_{l-1} + a_l + ……+a_r$

S[l-1]：前l-1个数的和，$s[l - 1] = a_1 +a_2+……+a_{l-1}$

S[r]-S[l-1]可以消去很多数，剩下的和就是区间[l, r]中所有数的和。

**前缀和可以用一次运算，求出任意一个区间内所有数的和**

# 本题解析

## 题目描述

输入一个长度为 n 的整数序列。

接下来再输入 m 个询问，每个询问输入一对 l,r。

对于每个询问，输出原序列中从第 l 个数到第 r 个数的和。

**输入格式**

第一行包含两个整数 n 和 m。

第二行包含 n 个整数，表示整数数列。

接下来 m 行，每行包含两个整数 l 和 r，表示一个询问的区间范围。

**输出格式**

共 m 行，每行输出一个询问的结果。

**数据范围**

1≤l≤r≤n
1≤n,m≤100000
−1000≤数列中元素的值≤1000

**输入样例：**

```
5 3
2 1 3 6 4
1 2
1 3
2 4
```

**输出样例：**

```
3
6
10
```

## C++代码

```C++
#include <iostream>

using namespace std;

const int N = 100010;
int a[N], s[N];

int main()
{
    int n, m;
    scanf("%d%d", &n, &m);
    
    for (int i = 1; i <= n; i ++ ) scanf("%d", &a[i]);
    
    for (int i = 1; i <= n; i ++ ) s[i] = s[i - 1] + a[i];
    
    while (m -- )
    {
        int l, r;
        cin >> l >> r;
        printf("%d\n", s[r] - s[l - 1]);
    }
    
    return 0;
}
```
