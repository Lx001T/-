# 题目描述

给定一个浮点数 n，求它的三次方根。

**输入格式**

共一行，包含一个浮点数 n。

**输出格式**

共一行，包含一个浮点数，表示问题的解。

注意，结果保留 6 位小数。

**数据范围**

−10000≤n≤10000

输入样例：

```
1000.00
```

**输出样例：**

```
10.000000
```

# 整体思路

数的三次方根是浮点数二分，相对与整数二分更简单。

## 思路

1. 先定义左右边界，由此求出mid = l + r >> 1
2. 要判断一个数 $x$ 的三次方根 $\sqrt[3]{x}$ 在哪个区间，用二分将区间从中点切开。
   - 比较$mid$与 $\sqrt[3]{x}$ 的大小，即$mid*mid*mid$与$x$的大小。
   - 如果$mid*mid*mid > x$，则x在中点左边。r = mid
   - 否则 l = mid
3. 最后输出l或者r都行。

## 注意事项

浮点数二分循环结束条件是区间做右端点的差值。

如果区间左右端点之间的差值足够小，二分结束。

**那这个差值到底要有多小才能代表二分结束呢？**

根据经验值，浮点数二分要保留t位小数，则差值比t多两位小数，即`1e-(t+2)`

如：结果保留6位小数，差值小于`1e-8`即可。

# 本题解析

## C++代码

```C++
#include <iostream>

using namespace std;

int main()
{
    double x;
    cin >> x;
    
    double l = -10000, r = 10000;
    while(r - l  > 1e-8)
    {
        double mid = (l + r) / 2 ;
        if (mid * mid * mid >= x) r = mid;
        else l = mid;
    }
    
    printf("%.6lf", l);
    
    return 0;
}
```
